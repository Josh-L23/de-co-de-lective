StudentPlanner

Born from our own struggles as students, StudentPlanner is a comprehensive web application designed to transform how students organize their academic lives. What started as a semester project evolved into a passion project that we're thrilled to share with the world.

This weekend marks the official launch of StudentPlanner - a tool created by students, for students. It represents our team's journey from learning JavaScript fundamentals to building a full-fledged application.
